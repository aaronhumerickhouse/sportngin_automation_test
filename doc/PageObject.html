def initialize(browser)
  $browser = browser
  expected_element if respond_to? :expected_element
  at? if respond_to? :at?
  has_expected_title? if respond_to? :has_expected_title?
end

+ (Object) expected_element(type, identifier, timeout = Watir.default_timeout)

Creates the expected_element? method to each page object. Waits until the the element from the the page object is present. If it doesn't, raise an exception.

Examples:

expected_element('link', {css: 'a.button.save', index: 0}, 15)

Parameters:

  • type (String) —

    The expected element type.

  • identifier (Hash) —

    Information to find the expected element.

  • timeout (Integer) (defaults to: Watir.default_timeout) —

    The length of time to wait before timeout.

32

34
35
 
# File 'src/pages/page_object.rb', line 31

def self.expected_element(type, identifier, timeout=Watir.default_timeout)
  define_method 'expected_element' do
    $browser.send("#{type.to_s}", identifier).wait_until_present timeout
  end
end
